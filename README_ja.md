# Discord Webhook Sender 🚀

DiscordのWebhookを利用して、メッセージを簡単かつ連続して送信するためのWebツールです。  
ブラウザだけで動作し、送信間隔や表示名、アイコンのカスタマイズも可能です。

[English README.md](README.md)

## ✨ 特徴

- **簡単送信**: Webhook URLを貼り付けるだけで即座に送信可能。
- **連続送信対応**: 指定した回数分、メッセージをループ送信できます。
- **高度なカスタマイズ**:
    - 送信間隔（ミリ秒単位）の調整。
    - Webhookの表示名（名前）とアイコンURLの一時的な変更。
    - `@everyone` メンションの自動付与。
- **進捗管理**: プログレスバーで送信状況をリアルタイムに確認。
- **エラーログ機能**: 送信に失敗した際、原因をログとして表示・ダウンロード可能。
- **自動保存**: 入力した設定はブラウザのLocalStorageに保存され、再訪問時に復元されます。

## 🚀 使い方

1. **Webhook URLを取得**: Discordの「サーバー設定」→「連携サービス」→「ウェブフック」からURLをコピーします。
2. **ツールに入力**: URL、メッセージ、回数、遅延時間などを入力します。
3. **送信開始**: 「送信」ボタンをクリックします。
4. **停止**: 途中で止めたい場合は「停止」ボタンを押してください。

## 🛠️ ファイル構成

- `index.html`: ツールの構造
- `style.css`: デザイン（レスポンシブ対応）
- `script.js`: 送信ロジック・保存機能

## ⚠️ 注意事項

- このツールは個人の効率化およびテストを目的としています。
- Discordの利用規約（ToS）を遵守し、**スパム行為や他者の迷惑になるような送信は絶対に行わないでください。**
- 短時間に大量のメッセージを送信すると、Discord側からレート制限（Rate Limit）を受けたり、Webhookが削除されたりする可能性があります。
